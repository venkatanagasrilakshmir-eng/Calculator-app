# Calculator-app

A modern Full Stack Calculator Application built using:

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Firebase


## Features

- Arithmetic Operations
- Responsive UI
- Calculation History
- Backend API
- Firebase Ready


## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Hosting
- GitHub Pages
- Render
